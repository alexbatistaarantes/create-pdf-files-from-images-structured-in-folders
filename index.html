Create PDF files from images structured in folders

The name says it all, but to exemplify, if you select the parent folder "Documents scans", from the following structure

  • Documents scans

    • Document 1

      • Image.jpg
      • Image (2).jpg
      • Image (3).jpg

    • Form

      • Form 1.png
      • Form 2.png

you will end up with the PDF files (and pages (in the same order)) below

    2. [Image (2).jpg]
    3. [Image (3).jpg]
